# Guide d’initiation à Git & Clonage du dépôt

## Introduction rapide à Git

Git est un système de gestion de versions.  
Quelques notions fondamentales :

- **Dépôt (repository)** : dossier qui contient tout l’historique des fichiers, modifications, et contributions.
- **Clone** : action de récupérer la copie d’un dépôt distant sur son ordinateur.
- **Commit** : enregistrement d’un ensemble de modifications.
- **Push / Pull** : envoyer ou récupérer les changements vers/depuis le serveur GitHub.

## Cloner ce dépôt

1. **Installer Git**  
 Suivez les instructions officielles :  
 [https://git-scm.com/downloads](https://git-scm.com/downloads)

2. **Ouvrir un terminal** et exécuter la commande suivante :

   ```bash
   git clone https://github.com/matthkowa/LSO_siril.git
   ```

3. **Accéder au dossier**  
   ```bash
   cd LSO_siril
   ```

Vous aurez alors accès à tous les scripts et fichiers du projet.
