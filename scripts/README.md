# Dossier des scripts SIRIL

Ce dossier contient les scripts utilisés pour le traitement d'images astrophotographiques avec SIRIL.

- **calibration/** : Scripts de calibration (offset, dark, flat).
- **empilement/** : Scripts pour l'alignement et l'empilement des images.
- **traitement/** : Scripts de traitement et post-process.
- **exemples/** : Exemples de scripts ou tests.

Ajoutez vos nouveaux scripts dans le dossier approprié pour faciliter l’organisation et la réutilisation.
